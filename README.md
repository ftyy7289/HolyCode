## 🛠 Features

HolyCode provides a focused environment for building software. It includes a suite of integrated tools designed to save time and reduce manual labor.

*   **Integrated Code Help:** Connect your existing AI subscriptions to assist with writing, debugging, and explaining code.
*   **Agent Orchestration:** Use multiple AI agents that work together to complete large tasks across your files.
*   **Headless Browser:** The software features a built-in browser helper that tests websites and gathers data automatically.
*   **Tool Collection:** Access over 30 built-in utilities for file management, data processing, and system operations.
*   **Privacy:** Keep your projects local. The software operates on your machine to ensure your code stays under your control.

## ⚙️ Installation steps

Follow these steps to prepare your system and run the program:

1.  **Check system requirements:** Your computer must run Windows 10 or 11. Ensure you have at least 8GB of RAM available for the AI services to run smoothly.
2.  **Run the installer:** Locate the file you saved in the previous step. Double-click the file to launch the installer. Follow the prompts on your screen.
3.  **Permissions:** Windows may show a security prompt because the app contains tools for developer tasks. Select "More info" and then "Run anyway" if the system tries to block the launch.
4.  **Initial launch:** Open the HolyCode icon from your desktop or start menu. The program will initialize its local database and agents. This process takes about one minute on the first run.
5.  **Configuration:** Enter your AI subscription keys in the settings menu. You can provide keys for Claude, OpenAI, or Gemini depending on your preferences.

## 🧱 Understanding the workstation

The main screen contains three primary areas. Understanding these areas helps you work faster. 

The left sidebar lists your projects. You can drag and drop folders from your computer into this window to start working. The central window acts as your main workspace where you interact with the code. The right sidebar shows your active agents. 

When you start a new task, select an agent from the right panel. Give the agent a command, such as "Fix the error in this file" or "Create a test for this web page." The agent reads your code and suggests changes immediately.

## 🚢 Using the headless browser

The headless browser is a powerful component. It allows the software to visit websites on your behalf. To use it, open the tool tray in the top menu and select "Browser Agent." You can provide a URL and ask the agent to extract specific information, such as page titles, headers, or form fields. 

This tool works well for testing your own applications. If you build a website, the browser agent can click buttons and fill out forms to ensure everything functions as you expect. You do not need to write code to perform these actions; the agent interprets your natural language commands.

## 📋 Troubleshooting common issues

If you have trouble with the software, try these steps:

*   **Connection errors:** Ensure your internet connection remains stable. Some agents require external access to verify your subscription keys.
*   **Performance lag:** Close other heavy applications if the software feels slow. AI processing uses system memory.
*   **Missing tools:** If a specific tool fails to run, check the settings panel to ensure your environment variables are set correctly for your system.
*   **Update checks:** Periodically visit the download link to see if a newer version is available. Older versions may lack performance improvements.

## 🛡 Security and privacy

HolyCode uses a local container approach. This means the engine runs in an isolated environment on your hardware. It does not send your private source code to external servers unless you specifically ask an AI agent to analyze it. You control the keys and the data. If you choose to disconnect your internet, the base tools of the application will continue to work for local file modifications.
